# Security Policy

## Supported versions

Security updates are provided for actively maintained default templates and workflow standards in this repository.

## Reporting a vulnerability

Do not open public issues for vulnerabilities.

Report privately to: `security@alpininsight.com`

Provide:

- Affected repository and component
- Impact summary
- Reproduction steps or proof of concept
- Suggested remediation if available

## Response targets

- Initial acknowledgment: within 3 business days
- Triage and severity assessment: within 7 business days
- Remediation timeline: based on severity and operational risk

## Disclosure

After remediation is available, maintainers coordinate disclosure timing with reporters and affected repository owners.
